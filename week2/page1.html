📗 ページ1：数・文字・true/false って？

データ型と typeof を覚えよう

進捗: 14.3% (1/7ページ)

🌟 詳細解説

JavaScript には数値・文字列・真偽値といった基本的な型があります。型を調べるには typeof を使います。

typeof 42;      // "number"
typeof "hello"; // "string"
typeof true;    // "boolean"

💡 課題

数値・文字列・真偽値を1つずつ宣言し、typeofで型を表示してみましょう。

✅ 模範解答

// 数値
let age = 25;
console.log(typeof age); // "number"

// 文字列
let name = "たろう";
console.log(typeof name); // "string"

// 真偽値
let isStudent = true;
console.log(typeof isStudent); // "boolean"
