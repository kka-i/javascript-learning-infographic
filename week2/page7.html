🌟 詳細解説

これまで学んだ条件分岐と繰り返しを組み合わせた典型的な練習問題が FizzBuzz です。

📌 FizzBuzz の流れ

  • 1から順番に数字を表示
  • 3の倍数なら Fizz
  • 5の倍数なら Buzz
  • 両方の倍数なら FizzBuzz

1から15までの数字を出力し、3の倍数はFizz、5の倍数はBuzz、両方の倍数はFizzBuzzと表示するプログラムを書いてみよう。

✅ 模範解答

for (let i = 1; i <= 15; i++) {
    if (i % 15 === 0) {
        console.log('FizzBuzz');
    } else if (i % 3 === 0) {
        console.log('Fizz');
    } else if (i % 5 === 0) {
        console.log('Buzz');
    } else {
        console.log(i);
    }
}
