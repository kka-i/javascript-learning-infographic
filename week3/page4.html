📙 ページ4：オブジェクトを使ってみよう

プロパティを追加・変更してみる

進捗: 66.7% (4/6ページ)

🌟 詳細解説

オブジェクトは後からプロパティを追加したり削除したりできます。

let person = { name: "太郎" };
person.age = 20;
delete person.name;

💡 課題

オブジェクトに grade プロパティを追加し、全プロパティを表示しましょう。

✅ 模範解答

let student = { name: "田中太郎", age: 17 };
student.grade = "高校2年";
for (let key in student) {
    console.log(key + ": " + student[key]);
}
