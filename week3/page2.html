📙 ページ2：配列を使ってみよう

配列メソッドを体験

進捗: 33.3% (2/6ページ)

🌟 詳細解説

push や pop など、配列の要素を追加・削除するメソッドを使ってみましょう。

let fruits = ["りんご", "みかん"];
fruits.push("バナナ");
console.log(fruits);
let last = fruits.pop();
console.log(last);

💡 課題

fruits に新しい果物を push し、全要素を表示しましょう。

✅ 模範解答

let fruits = ["りんご", "みかん"];
fruits.push("バナナ", "いちご");
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}
