📙 ページ3：オブジェクトってなに？

プロパティでデータをまとめよう

進捗: 50% (3/6ページ)

🌟 詳細解説

オブジェクトはキーと値の組み合わせでデータを管理する仕組みです。

let student = {
    name: "たろう",
    age: 16,
    grade: "高校1年"
};
console.log(student.name);

💡 課題

生徒オブジェクトを作り、name と age を表示してみましょう。

✅ 模範解答

let student = {
    name: "田中太郎",
    age: 17,
    grade: "高校2年"
};
console.log(student.name);
console.log(student.age);
