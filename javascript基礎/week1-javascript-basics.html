プログラミングの基礎とJavaScriptの基本構文

変数と定数
データを格納するための容器の概念を学びます。
// 変数の宣言 let userName = "田中太郎"; var age = 25; // 定数の宣言 const PI = 3.14159;

重要ポイント

  • let: ブロックスコープの変数
  • const: 再代入不可の定数
  • var: 関数スコープ（非推奨）
🔢
データ型
JavaScriptで扱える様々なデータの種類を理解します。
// 基本的なデータ型 let text = "文字列"; // String let number = 42; // Number let isActive = true; // Boolean let empty = null; // Null let notDefined; // Undefined

重要ポイント

  • 動的型付け言語
  • typeof演算子で型を確認
  • 型の自動変換に注意
🧮
演算子
計算や比較、論理演算を行うための記号を学習します。
// 算術演算子 let a = 10 + 5; // 加算 let b = 10 - 5; // 減算 let c = 10 * 5; // 乗算 let d = 10 / 5; // 除算 // 比較演算子 5 === 5 // 厳密等価 5 !== 3 // 厳密不等価

重要ポイント

  • ===と==の違い
  • 論理演算子（&&, ||, !）
  • 代入演算子（+=, -=, *=, /=）
🔀
条件分岐
条件によって処理を分岐させる方法を学習します。
// if文 let score = 85; if (score >= 90) { console.log("優秀です！"); } else if (score >= 70) { console.log("良好です"); } else { console.log("頑張りましょう"); } // switch文 switch (grade) { case "A": console.log("優秀"); break; default: console.log("普通"); }

重要ポイント

  • if-else文の基本構造
  • 三項演算子の使用
  • switch文でのbreak文の重要性
🔄
繰り返し処理
同じ処理を効率的に繰り返す方法を学習します。
// for文 for (let i = 0; i < 5; i++) { console.log("カウント: " + i); } // while文 let count = 0; while (count < 3) { console.log("実行中: " + count); count++; } // 配列の繰り返し let fruits = ["apple", "banana", "orange"]; for (let fruit of fruits) { console.log(fruit); }

重要ポイント

  • for文の基本構造
  • while文とdo-while文の違い
  • for...of文とfor...in文
🎯
練習問題
学習した内容を実際に使ってみましょう。
// 問題1: 1から10までの数字を表示 for (let i = 1; i <= 10; i++) { console.log(i); } // 問題2: 偶数のみを表示 for (let i = 1; i <= 20; i++) { if (i % 2 === 0) { console.log(i + " は偶数です"); } } // 問題3: 配列の合計値を計算 let numbers = [1, 2, 3, 4, 5]; let sum = 0; for (let num of numbers) { sum += num; } console.log("合計: " + sum);
