📘 ページ5：文字列の連結

文字列を組み合わせて新しい文字列を作る方法

進捗: 62.5% (5/8ページ)

🌟 詳細解説

文字列の連結とは、複数の文字列を繋げて一つの文字列にすることです。JavaScriptでは+演算子を使って文字列を連結できます。

🔗 文字列連結の仕組み

"Hello"
+
" World"
=
"Hello World"

📝 連結の基本ルール

  • 文字列同士を+で繋げることができます
  • 文字列と数値を+すると、数値も文字列として扱われます
  • スペースや記号も文字列として連結できます
  • 変数に入った文字列も連結できます
基本の連結 
let firstName = "太郎";
let lastName = "田中";
let fullName = lastName + firstName;
console.log(fullName); // "田中太郎"

// スペースを入れる場合
let fullNameWithSpace = lastName + " " + firstName;
console.log(fullNameWithSpace); // "田中 太郎"

✨ テンプレートリテラル（ES6）

💡 より便利な文字列連結

バッククォート（`）を使うテンプレートリテラルなら、${}の中に変数を書くだけで簡単に文字列を作れます。

テンプレートリテラル 
let name = "太郎";
let age = 20;

// 従来の方法
let message1 = "私の名前は" + name + "で、" + age + "歳です。";

// テンプレートリテラル（推奨）
let message2 = `私の名前は${name}で、${age}歳です。`;

console.log(message1); // "私の名前は太郎で、20歳です。"
console.log(message2); // "私の名前は太郎で、20歳です。"

⚠️ よくある間違い

❌ 間違い
let num1 = "5";
let num2 = "3";
let result = num1 + num2;
console.log(result); // "53" (文字列として連結される)
✅ 正しい
let num1 = 5;  // 数値
let num2 = 3;  // 数値
let result = num1 + num2;
console.log(result); // 8 (数値として計算される)
⚠️

注意点

文字列と数値を+で結合すると、数値も文字列として扱われます。計算したい場合は、両方とも数値型にしましょう。

💡 課題

変数を使って自己紹介文を作ってみよう。name、hobby、favoriteFoodの3つの変数を作り、テンプレートリテラルで「はじめまして、○○です。趣味は○○で、好きな食べ物は○○です。」という文を作成してください。

✅ 模範解答

JavaScript 
let name = "太郎";
let hobby = "読書";
let favoriteFood = "ラーメン";

let introduction = `はじめまして、${name}です。趣味は${hobby}で、好きな食べ物は${favoriteFood}です。`;

console.log(introduction);

📝 解説

テンプレートリテラルを使うことで、文字列の中に変数を簡単に埋め込むことができます。 ${変数名}の形で書くと、その部分が変数の値に置き換わります。
