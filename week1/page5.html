文字列の連結

文字列を組み合わせて新しい文字列を作る方法を学びます

文字列の連結とは？

"Hello"
+
" World"
=
"Hello World"

文字列の連結とは、複数の文字列を繋げて一つの文字列にすることです。JavaScriptでは+演算子を使って文字列を連結できます。

基本的な文字列連結

基本の連結 
let firstName = "太郎";
let lastName = "田中";
let fullName = lastName + firstName;
console.log(fullName); // "田中太郎"

// スペースを入れる場合
let fullNameWithSpace = lastName + " " + firstName;
console.log(fullNameWithSpace); // "田中 太郎"

テンプレートリテラル（ES6）

💡

より便利な文字列連結

バッククォート（`）を使うテンプレートリテラルなら、${}の中に変数を書くだけで簡単に文字列を作れます。

テンプレートリテラル 
let name = "太郎";
let age = 20;

// 従来の方法
let message1 = "私の名前は" + name + "で、" + age + "歳です。";

// テンプレートリテラル（推奨）
let message2 = `私の名前は${name}で、${age}歳です。`;

console.log(message1); // "私の名前は太郎で、20歳です。"
console.log(message2); // "私の名前は太郎で、20歳です。"

実践的な例

🏷️ 商品タグ作成

let productName = "ノートPC";
let brand = "Tech社";
let model = "Pro2024";

let tag = `${brand} ${productName} ${model}`;
console.log(tag); // "Tech社 ノートPC Pro2024"

📧 メール件名作成

let customerName = "田中様";
let orderNumber = "ORD-001";

let subject = `【重要】${customerName}のご注文${orderNumber}について`;
console.log(subject);

🌐 URL作成

let baseUrl = "https://api.example.com";
let endpoint = "users";
let userId = 123;

let fullUrl = `${baseUrl}/${endpoint}/${userId}`;
console.log(fullUrl);

よくある間違い

❌ 間違い
let num1 = "5";
let num2 = "3";
let result = num1 + num2;
console.log(result); // "53" (文字列として連結される)
✅ 正しい
let num1 = 5;  // 数値
let num2 = 3;  // 数値
let result = num1 + num2;
console.log(result); // 8 (数値として計算される)
⚠️

注意点

文字列と数値を+で結合すると、数値も文字列として扱われます。計算したい場合は、両方とも数値型にしましょう。
