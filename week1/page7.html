📘 ページ7：繰り返し処理

同じ処理を効率的に繰り返す方法を学びます

進捗: 87.5% (7/8ページ)

🌟 詳細解説

繰り返し処理（ループ）は、同じ処理を指定した回数だけ繰り返すことです。JavaScriptではfor文を使って繰り返し処理を書きます。

🔄 for文の基本構文

  • forキーワードで繰り返し処理を開始
  • 初期化、条件、更新の3つの部分で構成
  • 条件が真の間、波括弧内の処理を繰り返す
  • 効率的に同じ処理を複数回実行できる
for文の基本形 
for (初期化; 条件; 更新) {
    // 繰り返したい処理
}

// 具体例：1から5まで表示
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// 結果：1, 2, 3, 4, 5 が順番に表示される

💡 課題

for文を使って、1から10までの数値を表示するプログラムを作ってください。

✅ 模範解答

JavaScript 
for (let i = 1; i <= 10; i++) {
    console.log(i);
}

// より詳しい例：九九の表
for (let i = 1; i <= 9; i++) {
    for (let j = 1; j <= 9; j++) {
        console.log(i + " × " + j + " = " + (i * j));
    }
}

📝 解説

for文の括弧内は「初期化; 条件; 更新」の順で書きます。 i++はi = i + 1と同じ意味で、変数を1つずつ増やします。
