繰り返し処理

同じ処理を何度も実行する方法を学びます

繰り返し処理とは？

i = 1
→
処理実行
→
i = 2
→
処理実行
→
...

繰り返し処理とは、同じ処理を指定した回数だけ実行することです。JavaScriptではfor文を使って繰り返し処理を書きます。

基本的なfor文

for文の基本形 
for (初期値; 条件; 増減) {
    // 繰り返したい処理
}

// 具体例：1から5まで表示
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// 結果：
// 1
// 2
// 3
// 4
// 5
💡

for文の各部分の説明

  • 初期値：ループの開始時に1回だけ実行
  • 条件：この条件が真の間、ループが続く
  • 増減：1回ループするたびに実行

様々なfor文のパターン

🔢 0から始まるパターン

// プログラミングでよく使う形
for (let i = 0; i < 5; i++) {
    console.log(i);
}
// 結果：0, 1, 2, 3, 4

📈 2ずつ増加

// 偶数を表示
for (let i = 0; i <= 10; i += 2) {
    console.log(i);
}
// 結果：0, 2, 4, 6, 8, 10

📉 逆順（減少）

// カウントダウン
for (let i = 5; i >= 1; i--) {
    console.log(i);
}
console.log("発射！");
// 結果：5, 4, 3, 2, 1, 発射！

🔤 文字列との組み合わせ

// メッセージと回数を組み合わせ
for (let i = 1; i <= 3; i++) {
    console.log("第" + i + "章");
}
// 結果：第1章, 第2章, 第3章

実践的な例

🧮 掛け算の練習

// 3の段を表示
for (let i = 1; i <= 9; i++) {
    let result = 3 * i;
    console.log("3 × " + i + " = " + result);
}

📊 合計の計算

// 1から10までの合計
let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum = sum + i;
}
console.log("合計：" + sum);

🎵 歌詞の生成

// 「○匹の子豚」の歌
for (let i = 3; i >= 1; i--) {
    console.log(i + "匹の子豚が");
}
console.log("みんなで踊った");

for文とif文の組み合わせ

🔥

条件付きの繰り返し

for文の中でif文を使うと、条件に合う場合だけ処理を実行できます。

for文 + if文 
// 1から10までの偶数だけ表示
for (let i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        console.log(i + "は偶数です");
    }
}

// 3の倍数の時だけ特別なメッセージ
for (let i = 1; i <= 15; i++) {
    if (i % 3 == 0) {
        console.log(i + "：3の倍数！");
    } else {
        console.log(i);
    }
}
