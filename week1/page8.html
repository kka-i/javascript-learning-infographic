復習とチャレンジ

第1週目で学んだことを振り返り、総合的な課題に挑戦します

🎉 第1週目で学んだこと

📝

console.log()

メッセージや計算結果をコンソールに表示する

console.log("こんにちは");
🧮

四則演算

+、-、*、/ を使った計算

console.log(10 + 5); // 15
📦

変数（let）

データを保存する箱

let name = "太郎";
console.log(name);
🔗

文字列の連結

+ を使って文字列をつなげる

let message = "こんにちは、" + name + "さん";
🔀

条件分岐（if文）

条件によって処理を変える

if (age >= 18) {
    console.log("成人です");
}
🔄

繰り返し（for文）

同じ処理を何度も実行

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

🎨 クリエイティブチャレンジ

🎯 パターン描画プログラム

for文を使って面白いパターンを作ってみましょう：

星のピラミッド

// 星のピラミッド
for (let i = 1; i <= 5; i++) {
    let stars = "";
    for (let j = 1; j <= i; j++) {
        stars = stars + "★";
    }
    console.log(stars);
}

数字のピラミッド

// 数字のピラミッド
for (let i = 1; i <= 5; i++) {
    let numbers = "";
    for (let j = 1; j <= i; j++) {
        numbers = numbers + j;
    }
    console.log(numbers);
}
