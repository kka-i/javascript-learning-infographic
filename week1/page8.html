🌟 第1週目の復習

第1週目では、JavaScriptの基本的な概念を学びました。ここで学んだ内容を振り返ってみましょう。

📚 学習した内容

  • JavaScriptとは何か、どこで動くか
  • console.logで文字を表示する方法
  • 四則演算（+、-、*、/）の使い方
  • 変数（let）でデータを保存する方法
  • 文字列の連結とテンプレートリテラル
  • 条件分岐（if文）で処理を分ける方法
  • 繰り返し処理（for文）で効率的にコードを書く方法

💡 総合課題

これまで学んだ内容を組み合わせて、1から10までの数値について、偶数なら「偶数」、奇数なら「奇数」と表示するプログラムを作ってください。

✅ 模範解答

JavaScript 
for (let i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        console.log(i + "は偶数です");
    } else {
        console.log(i + "は奇数です");
    }
}

// より高度な例：九九の表
for (let i = 1; i <= 9; i++) {
    let result = "";
    for (let j = 1; j <= 9; j++) {
        result += i + "×" + j + "=" + (i * j) + " ";
    }
    console.log(result);
}

📝 解説

この問題では、for文（繰り返し）、if文（条件分岐）、変数、文字列連結を組み合わせています。 %は余りを求める演算子で、偶数判定に使います。
