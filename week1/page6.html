📘 ページ6：条件分岐

条件によって処理を変える方法を学びます

進捗: 75% (6/8ページ)

🌟 詳細解説

条件分岐とは、「もし○○なら△△をする」という処理です。JavaScriptではif文を使って条件分岐を書きます。

📝 if文の基本構文

  • ifキーワードで条件分岐を開始
  • 括弧()の中に条件を書く
  • 波括弧{}の中に実行する処理を書く
  • 条件が真（true）の場合のみ処理が実行される
if文の基本形 
if (条件) {
    // 条件が真の時に実行される処理
}

// 具体例
let score = 85;

if (score >= 70) {
    console.log("合格です！");}

💡 課題

変数ageに年齢を代入し、18歳以上なら「成人です」、18歳未満なら「未成年です」と表示するプログラムを作ってください。

✅ 模範解答

JavaScript 
let age = 20;

if (age >= 18) {
    console.log("成人です");
} else {
    console.log("未成年です");
}

📝 解説

if-else文を使うことで、条件によって異なる処理を実行できます。

🖥️ 条件分岐を試そう

Conditional Statements
> if文を使って条件分岐を試してみましょう
>

⭐ 重要ポイント

🔀

条件分岐で処理を制御

if文で条件に応じて処理を変えることができます。

🔍

比較演算子を活用

==、!=、>、<などで条件を作ります。
