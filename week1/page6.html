条件分岐

条件によって処理を変える方法を学びます

条件分岐とは？

開始
↓
条件をチェック
真（true）
処理A
偽（false）
処理B

条件分岐とは、「もし○○なら△△をする」という処理です。JavaScriptではif文を使って条件分岐を書きます。

基本的なif文

if文の基本形 
if (条件) {
    // 条件が真の時に実行される処理
}

// 具体例
let score = 85;

if (score >= 70) {
    console.log("合格です！");
}

// 年齢の例
let age = 20;

if (age >= 18) {
    console.log("成人です");
}

比較演算子

==（等しい）

let name = "太郎";
if (name == "太郎") {
    console.log("こんにちは、太郎さん！");
}

!=（等しくない）

let weather = "雨";
if (weather != "晴れ") {
    console.log("傘を持って行きましょう");
}

>（より大きい）

let price = 1000;
if (price > 500) {
    console.log("高い商品です");
}

<（より小さい）

let stock = 3;
if (stock < 5) {
    console.log("在庫が少なくなっています");
}

>=（以上）

let score = 80;
if (score >= 80) {
    console.log("優秀な成績です！");
}

<=（以下）

let speed = 40;
if (speed <= 50) {
    console.log("制限速度内です");
}

if-else文

💡

条件が偽の場合の処理

elseを使うと、条件が偽（false）の時の処理も書けます。

if-else文 
let score = 65;

if (score >= 70) {
    console.log("合格です！");
} else {
    console.log("不合格です。もう一度頑張りましょう！");
}

// 年齢による判定
let age = 16;

if (age >= 18) {
    console.log("成人です");
} else {
    console.log("未成年です");
}

実践的な例

🌡️ 気温による服装アドバイス

let temperature = 15;

if (temperature >= 25) {
    console.log("半袖がおすすめです");
} else {
    console.log("長袖を着ましょう");
}

🎯 テストの成績判定

let testScore = 85;

if (testScore >= 90) {
    console.log("優秀！");
} else {
    console.log("もう少し頑張りましょう");
}

🛒 在庫チェック

let stock = 2;

if (stock > 0) {
    console.log("購入できます");
} else {
    console.log("在庫切れです");
}
